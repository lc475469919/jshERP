# yize-erp

易泽 ERP 是一套从零开始开发的单公司制造业 ERP。

当前仓库只保留蓝图和实施计划，业务代码已清空。后续开发严格按蓝图阶段推进，不跳过系统底座和基础资料。

## 当前阶段

第 0 阶段：技术选型和系统底座。

已确认：

- 技术栈
- 项目目录
- 数据库规范

下一步落地：

- 登录
- 用户
- 角色
- 权限
- 菜单
- 字典
- 单据编号规则

## 暂不开发

在第 0 阶段和基础资料完成前，不开发生产、采购、销售、库存、工资等业务单据。
