# jshERP 微信小程序

这是基于现有 jshERP 后端接口创建的微信小程序端 MVP。

## 已实现

- 账号密码登录
- 验证码登录流程
- 首页
- 商品库存查询
- 我的信息与退出登录

## 开发运行

1. 打开微信开发者工具。
2. 导入目录：`/Users/mac/jshERP/jshERP-miniprogram`
3. 本地调试时，在开发者工具中关闭“校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”。
4. 模拟器可使用默认服务地址：`http://localhost:9999/jshERP-boot`
5. 真机调试时，把登录页“服务地址”改成电脑局域网 IP，例如：`http://192.168.3.42:9999/jshERP-boot`

默认测试账号：`jsh`

默认测试密码：`123456`
