# 实施路线

当前决定：项目从零开始，先做系统底座和基础资料。生产模块、采购、销售、库存、工资等业务单据必须等基础资料完成后再开发。

## 第 0 阶段：产品和骨架

- 确定系统名称和品牌
- 完成模块蓝图
- 完成数据库初稿
- 搭建后端工程
- 搭建电脑端工程
- 搭建微信小程序工程
- 建立登录、权限、菜单、字典、编号规则

## 第 1 阶段：基础资料

- 商品资料
- 商品分类
- 单位
- 客户
- 供应商
- 仓库
- 员工
- 部门
- 账户

## 第 2 阶段：进销存闭环

- 采购订单
- 采购入库
- 销售订单
- 销售出库
- 调拨
- 盘点
- 库存查询
- 应收应付

## 第 3 阶段：生产自制闭环

- BOM 管理
- BOM 模板
- 工序管理
- 不良品项
- 生产任务
- 生产领料
- 用料登记
- 工序汇报
- 生产质检
- 成品入库
- 倒冲领料
- 完工校验

## 第 4 阶段：委外闭环

- 委外任务
- 委外物料出库
- 委外提请付款
- 委外成品质检
- 委外成品入库
- 委外付款情况

## 第 5 阶段：工资和考勤

- 打卡
- 请假/加班/调休
- 考勤月报
- 计时工资
- 计件工资
- 工资单
- 工资发放

## 第 6 阶段：报表和看板

- 经营看板
- 生产看板
- 生产成本统计
- 订单进度跟踪
- 委外进度跟踪
- 物料追踪
- 缺料追踪
- 销售/采购/库存/财务报表

## 第 7 阶段：小程序补齐

- 移动审批
- 扫码出入库
- 生产任务查看
- 工序汇报
- 生产质检
- 打卡考勤
- 工资条
