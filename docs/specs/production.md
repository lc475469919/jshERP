## 自制业务

### 生产任务

- 初步定位：自制生产主单据，承接销售订单、库存补货或手工创建的生产需求，驱动领料、用料、工序汇报、质检和成品入库。
- 初步列表：生产任务编号、来源单据、产品、规格、计划数量、已领料数量、已汇报数量、已质检数量、已入库数量、计划开工日期、计划完工日期、生产部门、负责人、任务状态、审核状态、备注。
- 初步筛选：任务编号、产品编号/名称、来源单号、生产部门、负责人、日期范围、任务状态、审核状态。
- 初步动作：新增、修改、复制、删除、提交、审核、反审核、关闭、打印、导入、导出、生成领料、查看进度、查看关联单据。
- 初步状态：草稿、待审核、已审核、生产中、已完工、已关闭、已作废；状态名称需以参考页面为准。
- 后续需要采集：表单字段、明细行字段、是否支持多产品任务、来源销售订单规则、BOM 展开规则、工序路线规则、编号规则、关闭/作废条件、打印模板、权限点。

### 生产领料

- 初步定位：按生产任务和 BOM 需求生成原材料领用单，联动库存出库。
- 初步列表：领料单号、生产任务号、产品、领料仓库、材料、规格、应领数量、实领数量、单位、领料人、领料日期、审核状态、出库状态。
- 初步筛选：领料单号、生产任务号、产品、材料、仓库、领料人、日期范围、审核状态、出库状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、生成出库/确认出库、打印、导出、查看库存、查看任务。
- 初步状态：草稿、待审核、已审核、已出库、部分出库、已作废；库存影响状态需采集确认。
- 后续需要采集：是否由任务一键生成、超领/补领/退料规则、批号/序列号处理、仓库选择规则、成本归集方式、与库管出库单的关联字段。

### 用料登记

- 初步定位：登记实际消耗材料，用于修正生产领料与实际用料差异，并参与成本统计。
- 初步列表：登记单号、生产任务号、产品、材料、计划用量、实际用量、损耗数量、登记人、登记日期、审核状态。
- 初步筛选：登记单号、生产任务号、产品、材料、登记人、日期范围、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、导入、导出、查看任务、查看领料。
- 初步状态：草稿、待审核、已审核、已作废。
- 后续需要采集：用料登记是否必须依赖领料单、是否可多次登记、损耗项字段、差异处理方式、是否反写生产任务进度和成本。

### 工序汇报

- 初步定位：按生产任务和工序登记加工进度、人工、工时、合格数和不良数。
- 初步列表：汇报单号、生产任务号、产品、工序、员工/班组、汇报数量、合格数量、不良数量、工时、计件数量、汇报日期、审核状态。
- 初步筛选：汇报单号、任务号、产品、工序、员工/班组、日期范围、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、导入、导出、生成质检、查看工资、查看工序记录。
- 初步状态：草稿、待审核、已审核、已作废。
- 后续需要采集：工序是否按顺序控制、是否允许跳工序、计件工资生成规则、计时工资生成规则、工序汇报与生产质检的关系、移动端是否需要汇报入口。

### 生产质检

- 初步定位：对自制生产过程或完工产品进行质检，记录合格、不良及处理结果。
- 初步列表：质检单号、生产任务号、产品、工序/检验环节、报检数量、合格数量、不良数量、不良品项、处理方式、质检员、质检日期、审核状态。
- 初步筛选：质检单号、任务号、产品、工序、质检员、日期范围、不良品项、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、生成入库、返工登记、报废登记、打印、导出。
- 初步状态：草稿、待审核、已审核、部分合格、全部合格、不合格、已作废；具体状态需采集确认。
- 后续需要采集：质检字段、抽检/全检规则、不良品项选择、返工和报废流程、是否控制成品入库数量、是否生成库存报损。

### 成品入库

- 初步定位：将自制完工且可入库的成品转入仓库，形成库存入库记录。
- 初步列表：入库单号、生产任务号、产品、规格、入库仓库、应入数量、实入数量、单位、入库人、入库日期、审核状态、库存状态。
- 初步筛选：入库单号、任务号、产品、仓库、入库人、日期范围、审核状态、库存状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、确认入库、打印、导出、查看质检、查看任务。
- 初步状态：草稿、待审核、已审核、已入库、部分入库、已作废。
- 后续需要采集：是否必须通过质检、批号/序列号生成规则、成本入账规则、是否允许超任务入库、与库管入库单的关联方式。

## 委外业务

### 委外任务

- 初步定位：委外加工主单据，记录外协供应商、加工产品、计划数量和加工进度。
- 初步列表：委外任务编号、供应商、产品、规格、计划数量、已出库数量、已质检数量、已入库数量、加工费用、计划交期、负责人、任务状态、审核状态。
- 初步筛选：任务编号、供应商、产品、负责人、日期范围、任务状态、审核状态。
- 初步动作：新增、修改、复制、删除、提交、审核、反审核、关闭、打印、导入、导出、生成物料出库、提请付款、查看进度。
- 初步状态：草稿、待审核、已审核、委外中、已完工、已关闭、已作废。
- 后续需要采集：委外加工子入口的实际页面、供应商选择规则、加工费计价方式、是否关联销售订单/BOM、关闭条件、付款节点规则。

### 物料出库

- 初步定位：按委外任务向外协供应商发出原材料，联动库存出库。
- 初步列表：出库单号、委外任务号、供应商、材料、规格、仓库、应出数量、实出数量、出库人、出库日期、审核状态、出库状态。
- 初步筛选：出库单号、委外任务号、供应商、材料、仓库、日期范围、审核状态、出库状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、确认出库、打印、导出、查看库存、查看委外任务。
- 初步状态：草稿、待审核、已审核、已出库、部分出库、已作废。
- 后续需要采集：材料出库是否自动按 BOM 展开、补发/退回规则、批号/序列号处理、委外在途材料追踪口径。

### 提请付款

- 初步定位：从委外任务或委外入库节点发起应付加工费申请，后续对接财务付款。
- 初步列表：付款申请号、委外任务号、供应商、加工产品、申请金额、已付金额、申请人、申请日期、付款状态、审核状态。
- 初步筛选：申请号、委外任务号、供应商、申请人、日期范围、付款状态、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、生成付款单、打印、导出、查看委外任务、查看付款情况。
- 初步状态：草稿、待审核、已审核、待付款、部分付款、已付款、已作废。
- 后续需要采集：申请金额来源、是否按入库/质检数量结算、税率和发票字段、与财务付款单的关系、反审核限制。

### 成品质检

- 初步定位：对委外加工返回的成品进行质检，决定可入库数量和不良处理。
- 初步列表：质检单号、委外任务号、供应商、产品、报检数量、合格数量、不良数量、不良品项、质检员、质检日期、审核状态。
- 初步筛选：质检单号、委外任务号、供应商、产品、质检员、日期范围、不良品项、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、生成入库、退回供应商、打印、导出。
- 初步状态：草稿、待审核、已审核、部分合格、全部合格、不合格、已作废。
- 后续需要采集：是否支持供应商退回/返工、是否扣减委外进度、质检不良项字段、与成品入库数量控制的关系。

### 成品入库

- 初步定位：将委外加工合格成品入库，形成库存入库记录并推进委外任务完成度。
- 初步列表：入库单号、委外任务号、供应商、产品、入库仓库、应入数量、实入数量、入库人、入库日期、审核状态、库存状态。
- 初步筛选：入库单号、委外任务号、供应商、产品、仓库、日期范围、审核状态、库存状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、确认入库、打印、导出、查看质检、查看付款。
- 初步状态：草稿、待审核、已审核、已入库、部分入库、已作废。
- 后续需要采集：是否必须经过委外质检、加工费成本入库口径、材料成本结转口径、批号/序列号规则、与库管入库单的关联方式。

### 委外付款情况

- 初步定位：查看委外任务维度的应付、已付、未付和付款明细。
- 初步列表：委外任务号、供应商、产品、加工数量、加工金额、已申请金额、已付款金额、未付款金额、最近付款日期、付款状态。
- 初步筛选：委外任务号、供应商、产品、日期范围、付款状态。
- 初步动作：查看明细、导出、跳转付款申请、跳转付款单、打印。
- 初步状态：未申请、待付款、部分付款、已付款；状态需按参考页面核验。
- 后续需要采集：统计口径、是否按任务或供应商汇总、是否展示付款单明细、导出列、与财务应付账款的边界。

## 其他

### 工序管理

- 初步定位：维护生产工序基础资料，供生产任务、工序汇报、计件/计时工资引用。
- 初步列表：工序编号、工序名称、所属部门、默认计价方式、标准工时、计件单价、显示顺序、状态、备注。
- 初步动作：新增、修改、删除、启用、停用、导入、导出。
- 初步状态：启用、停用。
- 后续需要采集：参考页面筛选项、字段是否包含设备/岗位、工序顺序维护方式、是否有工序路线概念。

### BOM 管理

- 初步定位：维护成品与材料、工序、损耗和用量的生产配方。
- 初步列表：BOM 编号、成品、规格、版本、材料数量、工序数量、状态、生效日期、失效日期、备注。
- 初步动作：新增、修改、复制、删除、启用、停用、审核、反审核、导入、导出、生成模板。
- 初步状态：草稿、已审核、启用、停用、失效；需采集实际状态。
- 后续需要采集：表头字段、材料明细字段、工序明细字段、版本控制、替代料、损耗率、默认 BOM 规则、展开逻辑。

### BOM 模板

- 初步定位：维护可复用的 BOM 模板，用于快速创建具体成品 BOM。
- 初步列表：模板编号、模板名称、适用类别、材料数量、工序数量、状态、备注。
- 初步动作：新增、修改、复制、删除、启用、停用、导入、导出、套用模板。
- 初步状态：启用、停用。
- 后续需要采集：模板与正式 BOM 的转换规则、模板字段、是否支持按商品类别套用、权限点。

### 不良品项

- 初步定位：维护质检和工序汇报中使用的不良原因/不良项。
- 初步列表：不良项编号、不良项名称、适用环节、显示顺序、状态、备注。
- 初步动作：新增、修改、删除、启用、停用、导入、导出。
- 初步状态：启用、停用。
- 后续需要采集：是否分组、是否区分自制/委外、是否有责任部门和处理方式字段。

### 计时工资

- 初步定位：维护或登记按工时计算的生产工资，来源可能包括工序汇报或人工录入。
- 初步列表：工资单号、员工、部门、生产任务、工序、工时、时薪、工资金额、日期、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、导入、导出、汇总统计。
- 初步状态：草稿、待审核、已审核、已作废。
- 后续需要采集：是基础价格维护还是工资单据、与人事月度工资的关系、工时来源、审核流程、汇总口径。

### 计件工资

- 初步定位：维护或登记按合格件数计算的生产工资，来源可能包括工序汇报和计件单价。
- 初步列表：工资单号、员工、部门、生产任务、工序、合格数量、计件单价、工资金额、日期、审核状态。
- 初步动作：新增、修改、删除、提交、审核、反审核、导入、导出、汇总统计。
- 初步状态：草稿、待审核、已审核、已作废。
- 后续需要采集：计件单价来源、是否按工序/产品设价、不良数量是否计薪、与人事月度工资的关系、汇总口径。

### 工序汇报记录

- 初步定位：工序汇报的集中查询页，供追踪生产任务、员工产量和工资来源。
- 初步列表：汇报单号、任务号、产品、工序、员工/班组、汇报数量、合格数量、不良数量、工时、工资金额、日期、审核状态。
- 初步筛选：任务号、产品、工序、员工/班组、日期范围、审核状态。
- 初步动作：查看详情、导出、打印、跳转生产任务、跳转工资。
- 初步状态：沿用工序汇报状态。
- 后续需要采集：与“工序汇报”入口的差异、是否只读、是否支持反审核/删除、统计列。

## 生产报表

### 生产看板

- 初步定位：生产模块首页型看板，展示任务、进度、缺料、质检、入库和委外概览。
- 初步内容：任务数量、在制数量、完工数量、延期任务、缺料任务、今日汇报、今日入库、委外待回、异常不良。
- 初步动作：按卡片跳转明细、筛选日期/部门、刷新、导出或打印。
- 后续需要采集：实际卡片、图表类型、统计口径、默认日期范围、权限控制。

### 生产成本统计

- 初步定位：按生产任务、产品或期间统计材料、人工、委外和其他成本。
- 初步列表：任务号、产品、生产数量、材料成本、人工成本、委外成本、制造费用、总成本、单位成本、差异。
- 初步筛选：日期范围、产品、生产任务、部门、成本类型。
- 初步动作：查询、导出、查看明细、打印。
- 后续需要采集：成本归集来源、成本计算时点、是否区分计划/实际、委外成本结转口径。

### 订单进度跟踪表

- 初步定位：从销售订单或生产来源订单角度跟踪生产任务进度。
- 初步列表：订单号、客户、产品、订单数量、已下达任务数量、已领料、已汇报、已质检、已入库、未完成数量、交期、进度状态。
- 初步筛选：订单号、客户、产品、日期范围、进度状态。
- 初步动作：查询、导出、查看订单、查看生产任务。
- 后续需要采集：订单来源范围、进度百分比公式、延期判断规则。

### 委外进度跟踪表

- 初步定位：按委外任务跟踪发料、质检、入库和付款进度。
- 初步列表：委外任务号、供应商、产品、计划数量、已发料、已质检、已入库、应付金额、已付金额、交期、进度状态。
- 初步筛选：供应商、任务号、产品、日期范围、进度状态。
- 初步动作：查询、导出、查看委外任务、查看付款。
- 后续需要采集：进度计算口径、是否展示材料明细、付款与进度是否同表展示。

### 生产进度跟踪表

- 初步定位：按生产任务或工序维度跟踪自制加工进度。
- 初步列表：任务号、产品、计划数量、已领料、已汇报、合格数量、不良数量、已入库、当前工序、计划完工日期、进度状态。
- 初步筛选：任务号、产品、部门、工序、日期范围、进度状态。
- 初步动作：查询、导出、查看任务、查看工序汇报。
- 后续需要采集：进度百分比公式、工序级展开方式、延期和异常标记。

### 物料追踪表

- 初步定位：追踪生产任务所需物料的需求、领用、消耗和剩余。
- 初步列表：任务号、产品、材料、需求数量、已领数量、已用数量、退料数量、差异数量、仓库、批号。
- 初步筛选：任务号、产品、材料、仓库、日期范围。
- 初步动作：查询、导出、查看领料、查看用料登记、查看库存流水。
- 后续需要采集：是否覆盖委外物料、批号/序列号展示、差异计算方式。

### 缺料追踪表

- 初步定位：按生产任务和 BOM 需求检查库存缺口，辅助采购、调拨或补料。
- 初步列表：任务号、产品、材料、需求数量、可用库存、已领数量、缺料数量、建议补货数量、预计到货日期、缺料状态。
- 初步筛选：任务号、产品、材料、仓库、缺料状态、日期范围。
- 初步动作：查询、导出、生成请购/采购建议、查看库存、查看在途。
- 后续需要采集：缺料公式、可用库存口径、在途采购是否计入、是否支持一键补货。

### 计件工资统计表

- 初步定位：按员工、工序、任务或期间汇总计件工资。
- 初步列表：员工、部门、工序、产品、任务号、合格数量、计件单价、计件金额、日期范围、审核状态。
- 初步筛选：员工、部门、工序、产品、日期范围、审核状态。
- 初步动作：查询、导出、查看明细、打印、对接月度工资。
- 后续需要采集：汇总维度、是否按已审核汇报统计、是否扣除不良、与人事工资的结转方式。

### 计时工资统计表

- 初步定位：按员工、工序、任务或期间汇总计时工资。
- 初步列表：员工、部门、工序、任务号、工时、时薪、计时金额、日期范围、审核状态。
- 初步筛选：员工、部门、工序、日期范围、审核状态。
- 初步动作：查询、导出、查看明细、打印、对接月度工资。
- 后续需要采集：工时来源、时薪来源、加班/休息规则、与人事工资的结转方式。

## 共用业务关系

- 基础资料依赖：商品信息、商品属性、供应商、项目、仓库、员工、部门、物流公司。
- 生产资料依赖：BOM、BOM 模板、工序、不良品项、计件/计时工资规则。
- 库存联动：生产领料和委外物料出库减少库存；自制成品入库和委外成品入库增加库存。
- 财务联动：委外提请付款、委外付款情况对接应付账款和付款单；生产成本统计需要引用材料、人工、委外和入库成本。
- 人事联动：工序汇报、计件工资、计时工资后续对接员工月度工资。
- 报表联动：生产任务、委外任务、工序汇报、质检、入库、付款和库存流水共同形成进度、成本、物料和工资报表。

## 后续采集清单

- 逐页确认每个入口的标题、筛选条件、按钮工具栏、表格列、行操作、批量操作、分页和导出字段。
- 逐页确认新增/修改弹窗或页面字段、默认值、必填规则、校验规则、附件/图片、明细表格和选择器。
- 逐页确认状态字段、审核/反审核/关闭/作废/删除条件，以及状态对按钮显隐和库存/财务/工资影响的约束。
- 采集生产任务到领料、用料、汇报、质检、入库的完整链路，以及委外任务到物料出库、付款、质检、入库的完整链路。
- 采集 BOM 展开、缺料计算、成本统计、工资统计、进度跟踪和看板的公式口径。
- 采集打印模板、导入模板、导出字段、权限点和可能的小程序移动操作入口。

## 当前实现建议

第一阶段只建立菜单、路由占位和规格采集任务，不直接实现业务单据逻辑。

正式开发前，应先补齐基础资料中的仓库、员工、部门、客户、供应商、商品、BOM 和工序资料，再按“生产任务 -> 领料/出库 -> 汇报 -> 质检 -> 入库 -> 报表”的顺序实现。
