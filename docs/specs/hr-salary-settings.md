# 薪资设置规格

## 参考位置

- 菜单：人事 / 人事管理 / 薪资设置。
- 页面标题：员工薪资列表。

## 页面结构

页面采用列表维护结构，顶部为标题和说明，下方为员工薪资状态标签、筛选区、操作区、表格。

标签：

- 计薪员工。
- 不计薪员工。

筛选条件：

- 所属部门。
- 员工姓名。

工具栏：

- 修改。
- 导入。
- 导出。

表格列：

- 姓名。
- 状态。
- 基本工资(￥)。
- 岗位工资(￥)。
- 所属部门。
- 操作：修改。

弹窗/抽屉：

- 标题：薪资信息。
- 字段：姓名、所属部门、基本工资(￥)、岗位工资(￥)。

导入弹窗：

- 标题：薪资导入。
- 提示先下载固定模板。
- 提示不要修改模板表头。

## 当前实现

管理端已实现前端参考页面和菜单路由，路径为 `/hr/salary-settings`。

当前数据暂为页面内演示数据。后续进入人事模块正式阶段时，需要补充员工档案表、薪资档案表、导入接口、导出接口，并与月度工资、考勤、计件工资联动。
