# 参考系统登录与主框架规格

## 采集来源

- 参考系统：用户提供的制造业版体验环境。
- 采集范围：登录页、体验版本弹窗、登录后的主框架、首页看板、左侧菜单结构。
- clean-room 约束：只参考布局、交互、字段和业务入口，不复制源代码、图片、视频、Logo、品牌名称和下载素材。

## 登录页

页面为满屏登录入口，背景是动态视频/大图氛围层，主体居中约 1024px 宽，左侧为产品宣传信息，右侧为白色登录卡片。

右侧卡片宽度约 350px，圆角较小，包含账号登录和扫码登录两种模式：

- 账号登录标题：账号登录。
- 账号输入框占位：请输入账号/手机号。
- 密码输入框占位：请输入密码。
- 记住密码复选框。
- 忘记密码入口。
- 主按钮：立即登录。
- 企业微信登录入口。
- 卡片底部入口：体验一下、立即注册。

扫码登录模式包含：

- 标题：手机扫码 安全登录。
- 二维码区域。
- 提示：请打开 手机APP-我的 扫码登录。

体验弹窗包含五个版本：

- 标准版。
- 专业版。
- 制造业版：增强专业版，增加生产加工模块，适用前店后厂、销售生产一体化模式。
- 连锁版。
- 仓管版。

## 登录后主框架

参考系统登录后采用左侧一级菜单、顶部公司栏、页签栏、内容区和底部授权栏的结构。

左侧一级菜单：

- 固定宽度约 100px。
- 顶级模块纵向排列。
- 鼠标悬停/选中后弹出二三级菜单面板。
- 弹出面板按二级分类分列，每列显示分类标题和三级页面入口。

顶部栏：

- 左侧显示系统 Logo、公司名称、制造业版标识。
- 右侧显示新手向导、帮助中心、更多服务、当前用户和退出入口。

页签栏：

- 默认包含首页页签。
- 包含关闭操作下拉和刷新入口。
- yize-erp 当前用 Vue Router 渲染内容区，不使用 iframe，但保留参考系统的页签栏视觉和操作入口。

底部栏：

- 左侧显示服务到期、授权用户信息。
- 右侧显示版权信息。

## 首页看板

首页初始内容包含以下区域：

- 顶部业务卡片：销售单、采购单、生产任务、库存。
- 库存总额。
- 库存分布-仓库。
- 库存分布-品类。
- 客户销售排行榜。
- 销售业绩达成。
- 销售趋势。
- 我的待办：库存预警、待办出库、待办入库，以及采购订单、销售订单等。
- 快捷入口：采购管理、销售管理、生产管理、库存管理。

## 菜单结构

当前主菜单按参考系统制造业版菜单呈现，不再把临时的“系统管理/用户管理/角色管理”作为产品主入口展示。系统基础能力仍保留路由和接口，用于后续按参考系统中的账号、授权用户、权限相关入口重新整理。

顶级模块顺序：

- 采购。
- 销售。
- 生产。
- 库管。
- 物流。
- 财务。
- 总账。
- 客户。
- 人事。
- 基础。
- 设置。

当前已接入路由的参考入口：

- 基础 / 商品管理 / 商品信息。
- 基础 / 商品管理 / 商品属性。
- 基础 / 信息管理 / 供应商管理。
- 基础 / 信息管理 / 项目信息。
- 基础 / 信息管理 / 物流公司。

其余参考入口先呈现菜单和“待开发”提示，后续逐页采集后实现。
