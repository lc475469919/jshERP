## 对 yize-erp 的实施提醒

- 不能按菜单顺序直接开发。
- 必须先做系统底座，再做基础资料。
- 基础资料完成后，优先进销存闭环，再进入生产闭环。
- 生产模块中的计件工资、计时工资需要依赖员工、工序、工序汇报和工资规则。
