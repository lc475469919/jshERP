# 易泽 ERP

这是一个全新的单公司 ERP 项目，目标是重做一套属于自己的制造业 ERP。

项目参考成熟 ERP 的业务模块、流程、字段、状态和报表设计，但不复制第三方源码、图片、商标、数据库或受保护素材。所有代码、界面和品牌资源都在本项目中重新实现。

## 产品定位

- 单公司内部使用
- 电脑端管理后台
- 微信小程序移动端
- 后端 API 服务
- 进销存、财务、生产制造、委外、人事工资、考勤、报表一体化

## 目录结构

```text
yize-erp/
  apps/
    backend/       后端 API 服务
    admin-web/     电脑端管理后台
    miniprogram/   微信小程序
  docs/            产品、架构、模块和数据库设计
  packages/        共享类型、工具或组件
```

## 开发原则

- 按参考系统的功能范围重做，不受 jshERP 现有模块限制。
- 优先把业务闭环做完整，再逐步优化体验。
- 保留单公司模式，不做租户、套餐、到期续费、平台运营后台。
- 生产模块优先级最高，制造业版能力完整对标。
- 小程序不是附属品，要覆盖打卡、审批、扫码、生产汇报、库存和工资条等移动场景。

## 第一阶段目标

1. 建立完整产品蓝图。
2. 完成数据库模块设计。
3. 搭建后端、管理端、小程序工程骨架。
4. 先实现基础资料、商品、客户、供应商、仓库。
5. 再实现生产任务、BOM、领料、用料、工序汇报、质检、成品入库闭环。
